Menguji Kemampuan Penalaran Tingkat Lanjut

Tentang Benchmark Big-Bench Hard

Bagian ini didedikasikan untuk evaluasi model menggunakan Big-Bench Hard. Benchmark ini terdiri dari berbagai tugas yang menguji kemampuan penalaran tingkat lanjut dari model bahasa besar.

Kode Sumber: BigBenchHard.py 

from deepeval.benchmarks import BigBenchHard
from deepeval.benchmarks.tasks import BigBenchHardTask
from deepeval.models import OllamaModel

llm = OllamaModel(
    model="gemma3:1b"  # Replace with your model name
)
print(llm)


# Define benchmark with specific tasks and shots
benchmark = BigBenchHard(
    tasks=[BigBenchHardTask.BOOLEAN_EXPRESSIONS],
    n_problems_per_task=81
)

# Replace 'hf.co/CALISTA-INDUSTRY/gemma_3_1B_reasoning_multimodal_en_ft_v2' with your own custom model
benchmark.evaluate(model=llm)
print(benchmark.overall_score)

Penjelasan Kode Big-Bench Hard

1. Inisialisasi Model Ollama

Kode ini menginisialisasi model OllamaModel dengan nama model gemma3:1b. Anda dapat mengganti ini dengan nama model Ollama Anda sendiri.

2. Konfigurasi Benchmark Big-Bench Hard

Benchmark BigBenchHard dikonfigurasi untuk mengevaluasi tugas tertentu, dalam contoh ini BOOLEAN_EXPRESSIONS, dengan 81 masalah per tugas.

3. Evaluasi Model

Model dievaluasi menggunakan benchmark yang telah dikonfigurasi, dan skor keseluruhan kemudian dicetak.
